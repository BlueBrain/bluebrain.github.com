Detailed Description

Exposes the windows in a display group in JSON format.

This class also maintains a cache of thumbnails for all windows. The thumbnails are generated asynchronously when window are added.

Example client usage: GET /api/windows => 200 { "windows": [ {"title": "Title", "uuid": "abcd", ... } ] }

GET /api/windows/abcd/thumbnail => 200 data:image/png;base64-—IMAGE DATA-—
