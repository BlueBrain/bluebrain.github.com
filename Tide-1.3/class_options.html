Detailed Description

Rendering options which can be changed during runtime.

Can be serialized and distributed to the Wall applications, and also set as a Qml context object.

void Options::moveToThread ( QThread *  thread)

Move this object and its member QObjects to the given QThread.

This intentionally shadows the default QObject::moveToThread to include member QObjects which are stored using shared_ptr and thus can't be made direct children of this class.
