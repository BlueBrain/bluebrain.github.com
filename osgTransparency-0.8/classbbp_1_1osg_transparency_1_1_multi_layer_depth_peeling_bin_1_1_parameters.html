Parameters
slicesNumber of depth slices to use for depth peeling. A number between 1 and 8.
unprojectDepthsWhether use unprojected z values or not.
alphaAwarePartitionAdjust the quatiles for the depth partition based on alpha values (This only makes sense if slices >= 2).
superSamplingUnimplemented
opacityThresholdThe accumulated opacity at a fragment at which fragments behind can be considered completely occluded.

Default values for unspecified arguments are opacityThreshold: 0.99 splitPointQuantiles: [0.5] unprojectDepths: false alphaAwarePartition: false

The following environmental variables are looked up to override defaults: OSGTRANSPARENCY_REPROJECT_QUANTILES OSGTRANSPARENCY_ALPHA_AWARE_PARTITION OSGTRANSPARENCY_OPACITY_THRESHOLD OSGTRANSPARENCY_SPLIT_QUANTILES: A list of exact quantiles to use instead of even spacing.
