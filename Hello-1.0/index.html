About

Hello is a cross-platform toolkit, designed to run on any modern operating system, including all Unix variants. Hello uses CMake to create a platform-specific build environment. The following platforms and build environments are tested:

  • Linux: Ubuntu 12.10, 13.04, 13.10 RHEL 6 (Makefile, i386, x64)
  • Mac OS X: 10.8 (Makefile, XCode, i386, x64)

The API documentation can be found on bluebrain.github.io
