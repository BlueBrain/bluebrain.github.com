Detailed Description

Adapter for jsonrpc::Client to make JSON-RPC requests over an http::Client.

Example usage: http::Client httpClient; HttpCommunicator communicator{httpClient, "localhost:8888/jsonrpc"}; Client<HttpCommunicator> client{communicator}; auto response = client.request(...);
