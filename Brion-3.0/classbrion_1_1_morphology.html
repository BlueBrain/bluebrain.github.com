Detailed Description

Read access a Morphology file.

Following RAII, this class is ready to use after the creation and will ensure release of resources upon destruction.

brion::Morphology::Morphology ( const URI &  source)
explicit

Open the given source to a morphology file and parse it.

The actual data loading happens in a background thread and is lazily finalised in any get method. It is therefore faster to construct a set of morphologies and then read their data over a serial construct-and-read approach.

Parameters
sourceURI to load the morphology
Exceptions
std::runtime_errorif file is not a valid morphology file
