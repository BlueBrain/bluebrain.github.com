Detailed Description

Data mapping of a compartment report frame.

The mapping provides the number of compartments per neuron/section pair and the start offset of each neuron/section pair within a data frame. Mappings are view specific. The indices used to access the information for a neuron are not GIDs, but refer to the iteration order of the GIDSet from the view that provides tha mapping.
