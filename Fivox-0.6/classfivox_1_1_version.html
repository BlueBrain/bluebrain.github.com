static bool fivox::Version::check ( )
inlinestatic

Runtime check for ABI compatibility.

Call from code using LivreFivoxSource. Will fail if the executable was compiled against a version incompatible with the runtime version.

Returns
true if the link-time and compile-time DSO are compatible.
