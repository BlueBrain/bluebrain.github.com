4 High-level description of the main applications currently supported by Fivox.

6 ## Validated applications
7 
8 Real scientific use cases, coming from user requests and finished after a
9 collaboration with scientists:
10 
11 #### Local Field Potential
12 
13 ![](/doc/images/application_lfp.png "Local Field Potential")
14 
15 Compute the LFP signal at the extracellular space of a given circuit, based on
16 a simulation current report
17 ([more info](https://en.wikipedia.org/wiki/Local_field_potential)).
18 
19 The output is generated as a 3D volume, containing each of the voxels the LFP
20 value corresponding to the position at the center of the voxel.
21 
22 For completeness, a separate tool (sample-point) is provided, to generate the
23 time series of the output voltage at a specific 3D point. In this case the space
24 is not voxelized, and the computation is done for a single point, so the
25 resulting values are more accurate, as an exact 3D position is used as input
26 (as opposed to the voxelized space, in which several different points within a
27 same voxel take the same value). The output is a text file with one value per
28 timestep, being possible to plot this values in order to visualize the evolution
29 of the voltage over time as a 2D graph.
30 
31 
32 #### Voltage-Sensitive Dye
33 
34 ![](/doc/images/application_vsd.png "Voltage-Sensitive Dye")
35 
36 Generate the in-silico VSD optical imaging of a given circuit, based on the
37 compartment areas and simulation voltage report
38 ([more info](https://en.wikipedia.org/wiki/Voltage-sensitive_dye)).
39 
40 A separate tool is provided, with the ability to generate a 2D image with the
41 floating point values, representing the projection on the pial surface of the
42 brain; and also a 3D volume containing the VSD values for each of the
43 compartments (before the projection step).
44 
45 
46 #### Synapse distributions
47 
48 ![](/doc/images/application_synapses.png "Synapse distributions")
49 
50 Generate a 3D volume with the synapse distributions, with the count of synapses
51 per voxel, for the given pre and post-synaptic targets.

54 ## Potential applications
55 
56 Solutions that could be (or already are) implemented, but without a clear
57 application, as they don't fulfill any real scientific need:
58 
59 #### Spiking frequencies
60 
61 Generate a 3D volume with the spike frequencies per voxel, for the given
62 circuit and spike report.
63 
64 
65 #### Voltage fields
66 
67 Generate a 3D volume representing the influence of the compartment voltages in
68 the extracellular space (e.g. applying a square falloff) for a given circuit.
