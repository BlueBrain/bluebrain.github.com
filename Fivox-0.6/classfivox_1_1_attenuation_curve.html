float fivox::AttenuationCurve::getAttenuation ( const float  yCoord,
const bool  interpolate = false 
) const
inline

Get the attenuation for the given Y-coordinate.

If no curve file was loaded or the thickness has a negative value, returns 1.0f for any attenuation depth so it does not modify the value of event. If yCoord is larger than the thickness it returns the first element of the curve (generally 0.f)

Parameters
yCoordAbsolute position (Y axis) of the point.
interpolateIf true, interpolate the attenuation between the two closest indices in the attenuation curve.
Returns
the interpolated attenuation value according to depth; 1 if there is no attenuation curve or it is empty.
