Scene &  scene,
MeshLoader &  meshLoader 
)

Imports a file containing the positions and the filename of a list of meshes.

All loaded meshes are positioned at the corresponding coordinates. The file is a space separate ASCII and has the following format for every line:

x y z type filename

type is the kind of geometry that will be loaded (point, morphology or mesh)

Parameters
filenamename of the file containing the positions and geometry filenames
sceneScene holding the geometry
meshLoaderLoader used to load meshes
Returns
true if the file was successfully imported. False otherwise.
