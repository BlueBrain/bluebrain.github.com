Detailed Description

A class which tracks and reports progress of an operation/execution.

It reports on stdout using boost::progress_display and also reports to a user-defined callback.

It is safe to use in threaded environments like OpenMP for-loops. In that case, only the first OpenMP thread reports progress, but still tracks from all threads.
