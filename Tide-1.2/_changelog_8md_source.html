27 # Release 1.1 (30-06-2016)
28 
29 * [48](https://github.com/BlueBrain/Tide/pull/48):
30  Prevent users from opening a file multiple times.
31 * [47](https://github.com/BlueBrain/Tide/pull/47):
32  Loading and saving sessions no longer block the wall interface.
33 * [46](https://github.com/BlueBrain/Tide/pull/46):
34  Improved Launcher:
35  - The creation and caching of file thumbnails is more efficient.
36  - File thumbnails are displayed with their correct aspect ratio.
37  Session preview images are of better quality (no longer pixelated).
38 * [39](https://github.com/BlueBrain/Tide/pull/39):
39  Expose usage statistics to the REST interface for monitoring [DISCL-318].
40 * [38](https://github.com/BlueBrain/Tide/pull/38):
41  Fix a segfault that occured when opening grayscale images.
42 * [36](https://github.com/BlueBrain/Tide/pull/36):
43  The options can be modified from the Launcher panel.
44 * [34](https://github.com/BlueBrain/Tide/pull/34):
45  The options can be retrieved and modified through the REST interface.
46 * [30](https://github.com/BlueBrain/Tide/pull/30):
47  The contents can now be shown in fullscreen mode
48 * [28](https://github.com/BlueBrain/Tide/pull/28):
49  Simplify window interaction [DISCL-320].
50  This change makes interacting with contents more natural and intuitive:
51  - No differences between windows in presentation mode vs. regular mode
52  - A glow effect highlights windows which have the focus (capture touch
53  events).
54  - Regular windows always move by default; tapAndHold to get focus and zoom in.
55  - Webbrowsers and interactive streamers always have focus; move using borders.
56  - Resizing always preserves aspect ratio by default; tapAndHold any resize
57  handle to change the aspect ratio of compatible contents (images
58  and webbbrowsers).
59  - The one-to-one button (1:1) also resets the zoom level.
60  - Other minor improvements.
61 * [24](https://github.com/BlueBrain/Tide/pull/24):
62  Bugfix: Correctly setup Webbrowser proxy from 'http_proxy' ENV VAR
63 * [23](https://github.com/BlueBrain/Tide/pull/23):
64  Replaced the Dock with a new Qml Launcher and a side control panel to open it
65  [DISCL-313] and [DISCL-316]. The Launcher improves the user experience and
66  navigation with a grid view for files and sessions.
67 * [22](https://github.com/BlueBrain/Tide/pull/22):
68  Detect TapAndHold with the mouse
69 * [21](https://github.com/BlueBrain/Tide/pull/21):
70  Bugfix: avoid setting ContentSize to zero on empty deflect::SizeHints event
71 * [18](https://github.com/BlueBrain/Tide/pull/18):
72  Added side buttons to change PDF pages and navigate browser history
73 * [15](https://github.com/BlueBrain/Tide/pull/15):
74  Fix several issues with the touch events, including for Deflect streamers.
75 * [14](https://github.com/BlueBrain/Tide/pull/14):
76  Detect two-fingers swipe gestures [DISCL-356]. Swipe left and right to
77  navigate through PDF pages and Webbrowser history.
78 * [12](https://github.com/BlueBrain/Tide/pull/12):
79  Added a basic REST interface with support for open/load/save commands.
