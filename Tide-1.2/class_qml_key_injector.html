static bool QmlKeyInjector::send ( QInputMethodEvent *  inputEvent,
QQuickItem *  rootItem 
)
static

Send an InputMethodEvent (keyboard input) into an offscreen Qml scene.

This is a workaround for missing "active focus" support offscreen Qml applications. Users must be aware that the event will be delivered to the first QQuickItem encountered that has focus enabled, which may not be the expected behaviour.

Parameters
inputEventthe event to send
rootItemthe root item of the scene.
Returns
true if the event could be delivered
