Detailed Description

Bulk Synchronous Parallel system.

These functions implement the Bulk Synchronous Parallel programming model, in which message passing is synchronized as collective "supersteps". In this implementation, each superstep consists of sending out a set of messages and receiving all messages sent during that superstep. To this end, functions are provided for sending and receiving messages, and the user must specify when all its messages are done sending. BSP will then tell the user when the superstep is done, i.e. there are no more incoming messages to receive. The user should continue receiving messages until then.
