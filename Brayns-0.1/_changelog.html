master

  • #16 Introduced engines abstraction layer

  • #8 Fixes morphology loading. The data provided by the Brion API provides diameters, and the current implementation was considering radiuses.
  • #7 README update
  • #6 Added bounding box as a scene environment option
  • #5 Fixed alpha in recursive rays on transparent surfaces.
  • #4 Fixed Unit Tests and Alpha in ExtendedOBJRenderer
  • #1 Fixed alpha in OBJExtendedRenderer
