Detailed Description

The receiver side of the codash communicator pattern.

This receiver can use an existing Collage localNode for communication or it creates its own using the provided connection description. This class is intended to receive dash nodes + changes on them from a connected sender. This receiver maintains an own dash context used for holding received nodes.

Receive new changes from the connected sender.
