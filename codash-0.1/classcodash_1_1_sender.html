Detailed Description

The sender side of the codash communicator pattern.

This sender can use an existing Collage localNode for communication or it creates its own using the provided connection description. This class is intended to propagate dash nodes + changes on them to connected receivers. This sender maintains an own dash context used for tracking changes on the registered nodes.

The created local node is maintained by the sender. It will listen on the given connection description for incoming receivers. Note that the Collage library has to be initialized beforehand.
