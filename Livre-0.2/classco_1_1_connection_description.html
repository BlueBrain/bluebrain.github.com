CO_API bool co::ConnectionDescription::fromString ( std::string &  data)

Read the connection description from a string.

The string is consumed as the description is parsed. Two different formats are recognized, a human-readable and a machine-readable. The human-readable version has the format hostname[:port][:type] or filename:PIPE. The type parameter can be TCPIP, SDP, IB, MCIP, UDT or RSP. The machine-readable format contains all connection description parameters, is not documented and subject to change.

Parameters
datathe string containing the connection description.
Returns
true if the information was read correctly, false if not.
