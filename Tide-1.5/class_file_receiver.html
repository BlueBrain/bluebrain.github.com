Detailed Description

Receive HTTP file uploads.

Example client usage:

POST /api/upload { "filename": "cool image.png", "x": 20, "y": 50 } => 200 { "url" : "cool%20image.png" }

PUT /api/upload/cool%20image.png — BINARY DATA — => 201
