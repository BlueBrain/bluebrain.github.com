Detailed Description

Utility class to optimize the reception of any type of serialized data.

The buffer avoids memory reallocations for successive receive operations, which would otherwise be expensive for large data (such as image streaming).

The serialization::get<T>() function can be used with this buffer to directly extract the expected (copyable/movable) object stored in it.
