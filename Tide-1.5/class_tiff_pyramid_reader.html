Detailed Description

Reader for TIFF image pyramid files.

TIFF image pyramids start at level 0 (full resolution). The dimensions of each level above has half the dimensions of the level beneath it. The top of the pyramid is a single image, whose dimensions are equal to (or directly smaller than) the size of a single tile.

Example pyramid for an image of size (512x512) with tile size (128x128): level 2 /-\ <– top (128x128), 1 tile level 1 /—\ (256x256), 4 tiles level 0 /--—\ <– base (512x512), 16 tiles
