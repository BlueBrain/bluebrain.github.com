Detailed Description

The Configuration class manages all the settings needed by the Tide application.

Both WallConfiguration and MasterConfiguation extend this class to provide additional parameters specific to a Wall or Master process.

In a typical DisplayWall setup, the "screens" are configured to map 1 to 1 with the XServer screens (which can extend to one or more physical displays).

For testing purposes, the "screens" can also be represented as windows on a single display.

Note
For historic reasons, both the words "screen" and "tile" are used in xml configuration files and have the same meaning.
Warning
: this class can only be used AFTER creating a QApplication.
