1 # Tide
2 
3 ![Display wall running Tide](doc/wall.png)
4 
5 Tide is BlueBrain's *Tiled Interactive Display Environment*. It provides
6 multi-window, multi-user touch interaction on large surfaces - think of a giant
7 collaborative wall-mounted tablet. See Tide in action in this [youtube
8 video](https://www.youtube.com/watch?v=wATHwvRFGz0).
9 
10 ## Features
11 
12 Tide is a distributed application that can run on multiple machines to power
13 display walls or projection systems of any size.
14 
15 Its user interface is designed to offer an intuitive experience on touch walls.
16 It works just as well on non touch-capable installations by using its web
17 interface from any web browser.
18 
19 Tide helps users with:
20 * Presenting and collaborating on a variety of media such as high-resolution
21  images, movies and pdfs.
22 * Sharing multiple desktop or laptop screens using the
23  [DesktopStreamer](https://github.com/BlueBrain/Deflect/releases) application.
24 * Sketching new ideas by drawing on a whiteboard and browsing websites.
25 * Interacting with content streamed from remote sources such as high-performance
26  visualisation machines through the
27  [Deflect](https://github.com/BlueBrain/Deflect.git) protocol. In particular
28  all [Equalizer](https://github.com/Eyescale/Equalizer)-based applications
29  as well as [Brayns](https://github.com/BlueBrain/Brayns) ray-tracing engine
30  have built-in support.
31 * Viewing high-resolution, immersive [stereo 3D streams](https://git.io/fpsoQ)
32  on compatible hardware.
33 
34 ## Usage
35 
36 Run 'tide' from the bin folder to start the application. By default it launches
37 three wall processes in a horizonal layout. Content can be opened from the
38 control user interface or by opening the web interface at
39 [http://localhost:8888](http://localhost:8888).
40 
41 ![Tide master interface](doc/master_interface.png)
42 
43 On a touch wall, click the rocket icon on the left control bar to browse for
44 contents and applications. Touch points are received via the TUIO protocol on
45 port 1701. To simulate a touch interface, build and run the SimpleSimulator
46 application or download any client from [TUIO](http://www.tuio.org/?software).
47 On a smartphone, search for *TuioPad* app.
48 
49 For more detailed information refer to the
50 [User Guide](http://bluebrain.github.io/Tide-1.5/user_guide.html).
51 
52 ## Web Interface
53 
54 ![Tide web interface](doc/web_interface.png)
55 
56 The application can be entirely controlled from the web interface or even using
57 direct REST API calls. Simply drag and drop a file to upload it to the wall.

59 ## Building from Source
60 
61 Basic procedure on Linux / OSX:
62 
63  git clone --recursive https://github.com/BlueBrain/Tide.git
64  mkdir Tide/build
65  cd Tide/build
66  cmake -GNinja -DCLONE_SUBPROJECTS=ON ..
67  ninja
68 
69 The master branch always contains the latest development features; see
70 [Changelog](https://git.io/fpsK1).
71 To build a stable [release](https://github.com/BlueBrain/Tide/releases)
72 checkout its tag:
73 
74  git checkout 1.5.0
75 
76 Tide uses a set of [common CMake modules](https://github.com/Eyescale/CMake) to
77 help with the configuration and handling of dependencies. To install required
78 system packages on Ubuntu (apt-get) or OSX (macports) run:
79 
80  cmake .. -DINSTALL_PACKAGES=ON
81 
82 For detailed instructions on how to build Tide on a specific platform refer to
83 [Building Tide](http://bluebrain.github.io/Tide-1.5/building.html).

85 ## Similar Projects
86 
87 Tide is the direct successor of
88 [DisplayCluster](https://github.com/TACC/DisplayCluster) at BlueBrain. Most of
89 the code has been rewritten based on Qt's Qml2 engine and multithreaded
90 operations for greatly optimized performance and stability.
91 
92 It offers a modern alternative to [Sage/Sage2](http://sagecommons.org) built on
93 proven and efficient technologies (C++14, Qt5, MPI).
