1 # Brion
2 
3 ![](doc/BBPLOGO350.jpg)
4 
5 Welcome to Brion, a C++ library for read and write access to Blue Brain data
6 structures, including BlueConfig/CircuitConfig, Circuit, CompartmentReport,
7 Mesh, Morphology, Synapse and Target files.
8 
9 Brion can be retrieved by cloning the
10 [source code](https://github.com/BlueBrain/Brion.git).
11 
12 ## Features
13 
14 Brion provides the following major features:
15 
16 * Fast and low-overhead read access to:
17  * Blue configs (brion::BlueConfig)
18  * Circuit description (brion::Circuit)
19  * H5 Synapses data (brion::SynapseSummary, brion::Synapse)
20  * Target (brion::Target)
21  * BBP binary meshes (brion::Mesh)
22  * BBP H5 morphologies and SWC morphologies (brion::Morphology)
23  * Compartment reports (brion::CompartmentReport)
24  * Spike reports (brion::SpikeReport)
25 * Fast and low-overhead write access to:
26  * Compartment reports (brion::CompartmentReport)
27  * BBP binary meshes (brion::Mesh)
28  * BBP H5 morphologies (brion::Morphology)
29 * Basic [data types](@ref brion/types.h) to work with the loaded data using
30  [vmmlib](http://vmml.github.io/vmmlib),
31  [Lunchbox](http://eyescale.github.io/Lunchbox-1.8/index.html),
32  [Boost](http://www.boost.org/doc/libs).
33

46 ## About
47 
48 Brion is a cross-platform library, designed to run on any modern operating
49 system, including all Unix variants. Brion uses CMake to create a
50 platform-specific build environment. The following platforms and build
51 environments are tested:
52 
53 * Linux: Ubuntu 14.04 and RHEL 6 (Makefile, i386, x64)
54 
55 The [latest API documentation]
56 (http://bluebrain.github.io/Brion-1.5/index.html) can be found on
57 [bluebrain.github.io](http://bluebrain.github.io).
