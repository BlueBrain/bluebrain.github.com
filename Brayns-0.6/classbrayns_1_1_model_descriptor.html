Detailed Description

The ModelDescriptor struct defines the metadata attached to a model.

Model descriptor are exposed via the HTTP/WS interface.

  • Enabling a model means that the model is part of scene. If disabled, the model still exists in Brayns, but is removed from the rendered scene.
  • The visible attribute defines if the model should be visible or not. If invisible, the model is removed from the BVH.
  • If set to true, the bounding box attribute displays a bounding box for the current model
