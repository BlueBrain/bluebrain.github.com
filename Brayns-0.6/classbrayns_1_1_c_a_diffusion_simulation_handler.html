Detailed Description

The CADiffusionSimulationHandler class handles simulation frames for Calcium diffusion.

Frames are stored in ASCII files containing coordinates for CA atoms. Each frame is in a different file. The format of the frame is X Y Z values stored as text. For example: 215.388692 996.594668 338.199478
