Detailed Description

Group of attributes.

This structure contains all the attributes to be used in a certain context. Given a particular attributed_data object, all the attributes used with that object should come from the same attributes object, so that their indices are unique. The attributes object keeps track of the next unique index to assign to the next attribute that is added.
