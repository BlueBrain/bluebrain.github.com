Detailed Description

Attached attribute data system.

These data structures and functions allow the runtime attachement of attribute data to objects. Attributes have an array type, with basic types including boolean, integer, floating point (rational), and string (text).

Attributes are like members of a data structure, but they can be added and removed at runtime. The current mechanism for this involves an array of pointers to attribute data, with one entry for each defined attribute.

enum attribute_type

Enumerated attribute type identifier.

A type identifier for the basic type of the attribute. Available basic types are boolean (one bit), integer (word size), rational (word size), and text (one byte).
