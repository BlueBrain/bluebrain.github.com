Detailed Description

macros for bit manipulation

this header provides a set of macros for single-bit operations on integers, such as getting and setting individual bits Throughout this file, variables called power are related to variables called bit as power = 2^(bit). The power variables act as single-bit masks, and the bit variables are bit indices.
