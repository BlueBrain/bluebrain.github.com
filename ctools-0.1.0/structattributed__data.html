Detailed Description

Object attributed data.

This structure contains data attached to an object in the form described by attributes. This structure is responsible for memory management of this data, and thus has a destroy function. Attributed data is stored using a dynamically allocated array of pointers to dynamically allocated data. Each pointer represents a particular attribute, and points to the data for that attribute, or is NULL if that attribute is not attributed to this object. The size of the entries array is just big enough to include the data attributed, that is, the last non-NULL pointer.
