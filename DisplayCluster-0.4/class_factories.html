Detailed Description

A set of Factory<T> for all valid ContentTypes.

It is used on Wall processes to map Content objects received from the master application to FactoryObjects which hold the actual data. It implements a basic garbage collection strategy for FactoryObjects that are no longer referenced/accessed.
