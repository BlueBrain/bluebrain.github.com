bool WebServiceServer::addHandler ( const std::string &  pattern,
dcWebservice::HandlerPtr  handler 
)

Registers a request handler with a particular regular expression.

When the URL of an incoming request matches the regular expression the handler is invoked.

Parameters
patternA regular expression.
handlerA request handler. If the handler is a QObject, it should be moved to this thread before making any signal/slot connections.
Returns
true if the handler was registered succesfully, false otherwise, for instance if the regular expression is not valid.
