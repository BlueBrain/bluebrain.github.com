Detailed Description

An attribute of a dash Node.

Attributes store data in the DAG formed by nodes. All data access to an Attribute is thread-safe as long as it happens from different contexts. Modifying an Attribute is not visible to other contexts until a Commit is applied to the Context.

Attributes are by default only visible in the Context they are constructed in. They are made visible in other contexts using Context::map().
