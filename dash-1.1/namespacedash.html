Detailed Description

Data Access and Sharing.

The dash library uses a per-thread Context to provide an isolated view on the data stored in Node and Attribute. Nodes form a directed acyclyc graph and hold attributes. Attributes store any data. Modifications are migrated between contexts using a Commit.
