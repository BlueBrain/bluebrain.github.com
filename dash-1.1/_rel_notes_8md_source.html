6 # Introduction {#Introduction}
7 
8 Welcome to DASH, a C++ library for generic, efficient multi-threaded Data
9 Access and Sharing. This is a bugfix release for the initial release of DASH,
10 which provides the following features:
11 
12 * dash::Attribute providing generic data storage.
13 * dash::Node forming generic directed acyclic graphs and storing attributes
14 * dash::Context providing thread-safe access to attributes and nodes using
15  copy-on-write data protection
16 * dash::Commit to migrate data modifications from context to context
17

57 # About {#About}
58 
59 DASH is a cross-platform library, designed to run on any modern operating
60 system, including all Unix variants and the Windows operating system. DASH uses
61 CMake to create a platform-specific build environment. The following platforms
62 and build environments are tested:
63 
64 * Linux: Ubuntu 14.04 RHEL 6.5 (Makefile, i386, x64)
65 * Mac OS X: 10.9 (Makefile, XCode, i386, x64)
66 
67 The
68 [API documentation](http://bluebrain.github.io/dash-1.0/index.html)
69 can be found on [bluebrain.github.io](http://bluebrain.github.io/). As
70 with any open source project, the available source code, in particular
71 the [unit tests](https://github.com/BlueBrain/dash/tree/1.0/tests)
72 provide a reference for developing applications.
