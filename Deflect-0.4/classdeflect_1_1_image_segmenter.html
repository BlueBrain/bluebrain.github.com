DEFLECT_API bool deflect::ImageSegmenter::generate ( const ImageWrapper &  image,
const Handler &  handler 
) const

Generate segments.

The generation might be parallelized, calling the handler function concurrently in multiple threads for different segments. When one handler fails, the remaining handlers may or may not be executed.

Parameters
imageThe image to be segmented
handlerthe function to handle the generated segment.
Returns
true if all image handlers returned true, false on failure

const unsigned int  nominalSegmentHeight

Set the nominal segment dimensions.

If both dimensions are non-zero, the images will be devided as many segments of the desired dimensions as possible. If the image size is not an exact multiple of the segement size, the remaining segments will be of size: image.size % nominalSize.

Parameters
nominalSegmentWidthThe nominal width of the segments to generate (default: 0).
nominalSegmentHeightThe nominal height of the segments to generate (default: 0).
