Frequently asked questions
==========================

How do I report a problem?
--------------------------

If you find an issue while using RTNeuron, you can report it by
`opening an issue <https://github.com/BlueBrain/RTNeuron/issues/new>`__ on
GitHub.

(GPLv3). Refer to the
`license <https://github.com/BlueBrain/RTNeuron/blob/master/LICENSE.txt>`__
for details.
