LIVRECORE_API livre::TransferFunction1D::TransferFunction1D ( const std::string &  file)
explicit

Load transfer function from 1dt (ImageVis3D) file.

The content of the ASCII file consists of a first line with the number of sample points in the transfer function and then all the values in 'R G B A' format from [0.0f, 1.0f]. If the format is unspecified, float is used. If the file extension or format is not supported or the file could not be opened, a default transfer function is generated.

Parameters
filePath to the transfer function file.
