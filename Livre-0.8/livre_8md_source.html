5 Application Parameters:
6  --animation [=arg(=1)] (=0) Enable animation mode (optional frame delta for animation speed, use
7  --animation=-<int> for reverse animation)
8  --animation-fps arg (=0) Animation frames per second. By default (value of 0), it will request a new
9  frame as soon as the previous one is done
10  --animation-follow-data Enable animation and follow volume data stream (overrides --animation=value)
11  --frames arg (=[ 0 262143 ]) Frames to render [start end)
12  --num-frames arg (=4294967295) Maximum nuber of frames to render
13  --camera-position arg (=[ 0 0 1.5 ]) Camera position
14  --camera-lookat arg (=[ 0 0 0 ]) Camera orientation
15  --volume arg Memory dummy volume: mem://[?query parameters][#fragment]
16  with optional query parameters:
17  sparsity=<float>
18  datatype=(u)int(8,16,32), float
19  and optional fragment:
20  <width>,<height>,<depth>,<blocksize>
21 
22  Raw volume: [raw://]/filename.[raw|img|nrrd](?output=format)#1024,1024,1024(,inp
23  ut format)
24  with formats being one of: char, int8, unsigned char, uint8, short, int16,
25  unsigned short, uint16, int, int32, unsigned int, uint32, float
26  The default input format is uint8, the default output format is the input
27  format.
28 
29  Tuvok/UVF volume: [uvf://]/path/to/volume.uvf
30  --transfer-function arg .1dt transfer function file (from ImageVis3D)
31 
32 Volume Renderer Parameters:
33  --gpu-cache-mem arg (=3072) Maximum GPU cache memory (MB) - caches the texture data in GPU memory

35  --sse arg (=4) Screen space error
