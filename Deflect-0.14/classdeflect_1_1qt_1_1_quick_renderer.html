Detailed Description

Renders the QML scene from the given window using QQuickRenderControl onto the surface of the window or to an offscreen surface.

Note that this object needs to be moved to a seperate (render)thread to function properly.

Inspired by http://doc.qt.io/qt-5/qtquick-rendercontrol-window-multithreaded-cpp.html
