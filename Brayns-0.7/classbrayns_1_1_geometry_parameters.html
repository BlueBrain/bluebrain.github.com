bool brayns::GeometryParameters::getCircuitUseSimulationModel ( ) const
inline

Defines if a different model is used to handle the simulation geometry.

If set to True, the shading of the main geometry model will be done using information stored in a secondary model that contains the simulation information. See OSPRay simulation renderer for more details.
