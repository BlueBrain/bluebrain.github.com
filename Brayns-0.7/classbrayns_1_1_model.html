Detailed Description

The abstract Model class holds the geometry attached to an asset of the scene (mesh, circuit, volume, etc).

The model handles resources attached to the geometry such as implementation specific classes, and acceleration structures). Models provide a simple API to manipulate primitives (spheres, cylinders, triangle meshes, etc).
